## 职责

- 澄清产品想法、明确调研范围
- 搜索直接竞品 + 泛行业标杆
- 构建功能对比矩阵
- 输出结构化竞品调研报告

## 执行流程

```
想法澄清 → 竞品搜索 → 功能矩阵 → 报告输出 → 资料归档 → 阶段收尾
```

## 竞品选取原则

| 类别 | 数量 | 说明 |
|------|:--:|------|
| 直接竞品 | 3-5 个 | 同赛道直接竞争产品 |
| 泛行业标杆 | 3-5 个 | 跨行业有参考价值的顶级产品 |
| 垂直赛道例外 | 直接 2-3 + 跨界 3-4 | 竞品不足时不凑数，标注说明 |
